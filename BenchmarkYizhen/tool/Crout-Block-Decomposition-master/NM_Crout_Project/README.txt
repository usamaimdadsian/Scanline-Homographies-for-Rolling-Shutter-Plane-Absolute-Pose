-------------------------------------------------------------------------
Instructions on how to run Filip and Ksawery's Crout Decompistion Program
-------------------------------------------------------------------------

1. Make sure all 3 attached Matlab files are in the same directory or folder.
2. In Matlab, type: [A,b] = GenerateCroutMatrix(p); %where p is the size of the p x p inner Aij matrices(I usually choose 2 or 3). Whole matrix A is of size 3*p x 3*p
3. Then, execute this: x = SolveByCrout(A,b)
4. Look at unexpected beautiful output
